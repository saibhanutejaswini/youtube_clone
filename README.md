YouTube Clone (HTML & CSS)
A simple YouTube homepage clone built using HTML and CSS. This project replicates the basic layout of YouTube to practice frontend skills like layout design, styling, and responsiveness.

Features
YouTube-like navbar
Sidebar menu
Video grid layout
Video thumbnails with title & channel info
Clean and responsive design
Built using Flexbox

Technologies Used
HTML5
CSS3
Flexbox
Project Files
index.html
style.css
images/

 How to Run
Open index.html in any web browser.
 
 Purpose
Created to practice HTML structure, CSS styling, alignment, and responsive layout by recreating a real-world website UI.
